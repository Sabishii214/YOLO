YOLO Object Detection - Git Repository Information

Repository: Ultralytics YOLOv8
URL: https://github.com/ultralytics/ultralytics
Version: Latest (installed via pip)

Installation method:
python -m pip install ultralytics opencv-python

Model used: YOLOv8n (nano)
Model file: yolov8n.pt (auto-downloaded on first run)

Official Documentation: https://docs.ultralytics.com/

To run:
# In VS Code terminal (PowerShell)
cd "D:\VS Code\YOLO"
chmod +x scripts/run-detector.sh

cd "D:\VS Code\YOLO\docker"
bash build-docker.sh
# Wait until it says: “Docker image built successfully! Image name: yolo-detector:latest”

In VS Code terminal: from project root (D:\VS Code\YOLO)= change dir
docker run --rm -v "D:\VS Code\YOLO:/app" -w /app -it yolo-detector ./scripts/run-detector.sh

To run webcam:(q to exit)
python detect.py

Notes:
- YOLOv8 is installed as a Python package (ultralytics)
- Model weights are downloaded automatically on first run
- No need to clone the repository manually
- All functionality accessed via Python API
